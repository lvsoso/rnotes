    llm-mcp

    # Model Context Protocol

    MCP Host: IDE, 管理多个 MCP Server; MCP Server: 实现MCP服务接口,处理MCP Client的请求; MCP Client: 实现MCP Client接口;

    server 有多种语言的实现，可以使用uv或npx进行启动。
