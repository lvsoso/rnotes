    # 指标类型

    • Counter 单调递增，QPS？
    • Gauge 可增可减
    • Histogram
      • sum 所有观测值的和
      • count 观测的事件总数
      • bucket 对观测值桶的累积计数
    • summary
      • sum 所有观测值的和
      • count 观测的事件总数
      • basename{quantile="xxx"} 分位数值
    http_server_requests_seconds_sum{application="demo",method="GET",outcome="SUCCESS",status="200",uri="/hello"} 0.032
