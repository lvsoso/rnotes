###  批量处理
“将多个句子拼接到一块儿作为输入” 不等于 “批量处理”。

**批量处理** 一般说的是并行批量。

将多个相互独立的问题作为输入进行批量并行处理。

- 使用 bacth tokenizer，截断或者填充；
- 构造prompt序列；
- 生成；
- 需要等待所有的结果生成完毕了；

加快速度？生成的结果长度不会相差太大。
